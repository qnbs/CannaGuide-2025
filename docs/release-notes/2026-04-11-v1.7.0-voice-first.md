# CannaGuide v1.7.0 "Voice-First Edition"

**Release Date:** 2026-04-11
**Previous Version:** v1.6.3

---

## Highlights

### Voice Pro v1.8

- **Voice Orchestrator** -- Central state machine (IDLE/LISTENING/PROCESSING/SPEAKING/CONFIRMATION) with 37 voice commands covering all app features
- **Porcupine WASM Wake-Word** -- 11 built-in keywords, BYOK AccessKey, dual-engine support (regex/Porcupine), 100% on-device
- **ElevenLabs Cloud TTS** -- BYOK with AES-256-GCM encrypted API key at rest, rate-limited (5 req/min), 5 languages
- **speakNatural() Normalization** -- 30+ cannabis/science abbreviation expansions, markdown stripping, unit conversion
- **Voice Worker** -- Off-main-thread processing: filler removal (5 languages), 3-pass command matching (exact/Levenshtein/keyword), waveform computation
- **VoiceHUD** -- Floating overlay with AnalyserNode waveform visualization, mode indicator, confirmation controls, a11y (ARIA role="status")
- **Voice Telemetry** -- Opt-in anonymous analytics, ring buffer (500 events), no PII or transcript storage

### Equipment & Onboarding

- **12 Preset Setups** -- Micro Stealth to Full Spectrum Pro, category filtering, difficulty badges, copy-to-my-setups
- **Onboarding-Seeded Presets** -- Budget/space/goal choices auto-sort presets with "Recommended for You" badge

### Enhanced PDF Reports

- Metrics charts (height/CO2 line charts, severity bar charts)
- Diagnosis trend visualization, AI-generated plant summary
- Offline template fallback, metric statistics (min/max/avg)

### Mobile AI Optimization

- Battery gating (<20% forces eco mode), RAF-throttled streaming
- Cancelable WebLLM downloads, mobile-specific inference timeouts
- Storage quota checks, memory pressure monitoring (80% threshold)

### Knowledge Expansion

- Lexikon: 83 -> 91 entries (8 orphaned entries integrated, 9 new definitions)
- Diagnosis i18n migration: 33 zero-shot labels via i18n keys

### Accessibility & UX

- PhotosTab/ScreenshotGallery: Radix DialogWrapper (focus trap, escape, screen reader)
- Confirmation dialogs: Grow archive + bulk favorite removal
- Scroll-to-top on all view/tab transitions
- Empty state icons for MetricsOverviewTab and InlineStrainSelector
- ~34 hardcoded Tailwind colors replaced with theme-aware classes

### CI & Security

- SLSA Verifier CI (SC-01): automated post-release verification
- Release pipeline dry-run mode (SC-03)
- Lighthouse CI: 0.95+ (performance/accessibility/best-practices)
- CodeQL #280: voiceWorker origin check
- Dependabot #50: basic-ftp pinned to 5.2.2
