# CannaGuide v1.8.1

**Release Date:** 2026-04-14
**Previous Version:** v1.8.0

---

## Highlights

### Domain Type Consolidation

- 120+ types extracted from monolithic `apps/web/types.ts` to
  `@cannaguide/ai-core/src/domain/` (10 domain files + barrel)
- `apps/web/types.ts` reduced from 1869 to 583 lines via bridge
  re-exports

### AI & Local ML

- **Model Registry** in `@cannaguide/ai-core` -- 4 models with
  GPU-tier auto-selection, version tracking, deprecation flags
- **Cache key versioning** -- inference cache keys include
  `model@version` for automatic invalidation on model upgrades
- **Telemetry percentiles** (p50/p95/p99) and per-model tracking
- **Vision Auto White Balance** -- Gray World color correction
  in visionInferenceWorker for grow-tent LED casts
- **Critical battery gating** -- `<15%` forces cloud/heuristic
  routing with Sentry event tracking and UI notifications
- **OPFS model cache** -- PlantVillage model in Origin Private
  File System with IndexedDB fallback and management UI
- **Whisper-Tiny STT skeleton** (V-06) -- lazy pipeline loading,
  GPU mutex coordination, i18n keys in all 5 languages

### Hydroponic Enhancements

- **PDF grow reports** include hydroponic readings section
  (pH/EC/Temp min/max/avg summary table)
- **CSV export** includes hydroponic time-series data
- **Trend badge** upgraded to color-coded pill with per-level
  ring styling and `role="status"` accessibility

### Seed Vault Overhaul

- 5 new sub-components, extended data model (SeedSource,
  storageLocation, germinationRate, batchNumber, expiryEstimate)
- 3 new Redux actions, grid/list view, bulk operations, pollen log
- breedingSlice tests expanded from 6 to 30

### CI & Infrastructure

- **Cloudflare Pages** -- wrangler-action SHA pinned to v3.14.1,
  project-name corrected to `cannaguide-2025`
- **i18n CI** -- completeness check added to release-gate and
  deploy workflows
- **Crowdin** config populated for community translation management
- **Bundle budget** -- brotli enforcement in check-bundle-budget.mjs
- **Node.js** minimum raised from 20 to 24 (Active LTS)

### i18n Fixes

- Missing `errorBoundary.retry` key added to ES/FR/NL
- Missing CRDT keys added to ES settings
- Crowdin config ready for community translation onboarding
